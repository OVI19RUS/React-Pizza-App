Тестовое приложение "Интернет магазин" на React на готовое верстке (перенос из html в jsx).

Полный стек:
- ReactJS
- Redux
- redux-thunk
- React Router
- Axios
- json-server

Статичный файл-сервер содержит в себе 16 пицц, которые можно фильтровать по категориям, а также сортировать по алфавиту, цене и популярности.

Функционал полный - работают все кнопки (кроме оплаты, разумеется), анимации, стоимость и количество пицц, удаление/добавление пицц в корзине, выбор разных свойств пицц, полная очистка корзины, переходы по кнопкам без перезагрузки приложения.
